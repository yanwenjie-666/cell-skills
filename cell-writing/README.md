# cell-writing

> Cell Press 格式稿件各章节撰写 | Manuscript Section Drafting for Cell Press Format

## 功能 | Features

本技能帮助你按照 Cell Press 期刊规范撰写论文各个章节：

- 📝 Summary（≤150 词，无引用，无缩写）
- 💡 Highlights（3–4 条，每条 ≤85 字符）
- 📰 eTOC Blurb（~50 词通俗摘要）
- 📖 Introduction（漏斗结构，"Here, we..."结尾）
- 🔬 Results（声明式小标题，假说驱动叙事）
- 💬 Discussion（意义阐释，非重复结果）
- 🔧 STAR Methods（结构化、可复现）
- 📋 Author Contributions（CRediT 格式）
- ⚖️ Declaration of Interests

This skill drafts complete manuscript sections following Cell Press structure and formatting requirements.

## 适用场景 | Use Cases

- 从头撰写 Cell Press 格式论文各章节
- 将已有稿件重新组织为 Cell Press 结构
- 生成符合字数限制的 Summary / Highlights / eTOC Blurb
- 撰写声明式 Results 小标题
- 组织 STAR Methods 的规范化结构

## 使用方法 | Usage

```
"帮我写一个 Cell 格式的 Summary，研究内容是..."
"Draft a Results section for Cell Reports based on these findings..."
"Write Highlights for my paper about [topic]"

```

## Cell Press 独特结构 | Unique Structure

```
Title

Lead Contact
Summary (≤150 words)          ← 非 Abstract
Keywords
Highlights (3-4 × ≤85 chars)  ← Cell Press 独有
eTOC Blurb (~50 words)        ← Cell Press 独有
Graphical Abstract             ← 必需
Introduction
Results
Discussion
Acknowledgments
Author Contributions (CRediT)
Declaration of Interests       ← 非 Competing Interests
Figure Legends
STAR Methods                   ← 非 Methods
References
Supplemental Information
```
