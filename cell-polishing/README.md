# cell-polishing

> Cell Press 风格学术英文润色 | Academic Prose Polishing for Cell Press Style

## 功能 | Features

本技能将学术稿件润色为 Cell Press 期刊的独特写作风格：

- 🇺🇸 美式英语拼写与标点（color, analyze, tumor, serial comma）
- 📖 Cell 式叙事驱动写作风格（narrative-driven, hypothesis-oriented）
- ✍️ 主动语态优先，第一人称"We"
- 🔗 段落间逻辑衔接优化
- 📐 Cell Press 专有术语规范（Lead Contact, STAR Methods, eTOC Blurb 等）
- 🧬 基因/蛋白命名规范（斜体/罗马体）
- 📊 统计报告规范化

This skill polishes academic manuscripts to match the distinctive writing voice of Cell Press journals.

## 适用场景 | Use Cases

- 将非母语英文稿件润色至 Cell Press 投稿水平
- 英式英语→美式英语转换
- 加强论文叙事性和逻辑流
- 检查并修正 Cell Press 专有术语
- 优化句子结构和段落组织

## 使用方法 | Usage

```
"Please polish this paragraph for Cell Press style"
"帮我把这段润色成 Cell 的写作风格"
"Convert this text to American English and Cell journal tone"
"Check this Introduction for Cell Press conventions"
```

## Cell vs Nature 写作风格对比 | Style Comparison

| 维度 | Cell Press | Nature |
|------|-----------|--------|
| 语言 | 美式英语 | 英式英语 |
| 风格 | 叙事驱动，讲故事 | 简洁精炼，数据导向 |
| 语态 | 主动语态为主 | 被动可接受 |
| 过渡 | 丰富的逻辑连接 | 简短过渡 |
| 标题 | 声明式（陈述结论） | 描述式居多 |
| 人称 | "We" 标准用法 | "We" 可用 |
