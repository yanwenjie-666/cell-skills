# cell-response

> Reviewer Response Letters for Cell Press Journals | Cell Press 审稿人回复信撰写

---

## 功能 | Features

- 📝 逐条分析审稿意见（类型、优先级、策略）
- ✍️ 撰写 Point-by-Point Response Letter
- 📬 生成修改版 Cover Letter
- 📋 输出修改行动清单
- 🛡️ Appeal Letter（拒稿申诉信）模板
- 🎯 针对 Cell Press 编辑特点优化策略

---

## 适用期刊 | Target Journals

Cell、Molecular Cell、Cell Stem Cell、Cell Reports、Cell Systems、Cell Chemical Biology、Cell Host & Microbe、Immunity、Neuron、Developmental Cell、Current Biology 等。

---

## 快速使用 | Quick Start

```
请帮我分析以下审稿意见并撰写回复信：

[粘贴 reviewer comments / decision letter]
```

---

## 回复策略 | Response Strategies

| 策略 | 适用情况 | 关键短语 |
|------|---------|---------|
| Agree & Do | Reviewer 说得对，可执行 | "We have now..." |
| Partially Agree | 部分有理，需折中 | "While we agree... we note..." |
| Respectfully Disagree | Reviewer 理解有误 | "We respectfully maintain..." |
| Beyond Scope | 超出本文范围 | "beyond the scope of the current study" |

---

## 输出结构 | Output Structure

1. **审稿意见分析表** — 类型 + 优先级 + 策略
2. **完整回复信** — 逐条 point-by-point
3. **Cover Letter** — 修改摘要
4. **修改行动清单** — 稿件修改 TODO

---

## Cell Press 特殊注意 | Cell Press Notes

- 编辑是全职专业编辑（非学术兼职）
- 回复需单独回应 Editor 的独立意见
- "Essential Revisions" 必须全部解决
- 语气：专业、感恩、合作
