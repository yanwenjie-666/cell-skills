# Cell Press Citation Format Guide

## Overview

Cell Press journals use a numbered citation system with references listed in order of appearance in the text. This guide covers all Cell Press family journals including Cell, Molecular Cell, Cell Stem Cell, Cell Reports, Cell Systems, Immunity, Neuron, and others.

## In-Text Citation Rules

### Basic Format
- Citations appear as superscript numbers: text^1 or text^1,2,3
- Multiple citations are separated by commas: ^1,2,3
- Ranges use en-dash: ^1-3
- Citations appear after punctuation when at end of sentence

### Placement
- Place citation number immediately after the relevant claim
- When citing multiple sources for one claim, group them: ^4-6
- For author-named citations in text: "Smith et al.^7 showed that..."

## Reference List Format

### Journal Articles (most common)
```
Author1, A.A., Author2, B.B., and Author3, C.C. (Year). Title of article. Journal Name Volume, pages. DOI.
```

**Examples:**
```
1. Weinberg, R.A., and Hanahan, D. (2011). Hallmarks of cancer: the next generation. Cell 144, 646-674. https://doi.org/10.1016/j.cell.2011.02.013

2. Zhang, F., Cong, L., Ran, F.A., Cox, D., Lin, S., Barretto, R., Habib, N., Hsu, P.D., Wu, X., Jiang, W., et al. (2013). Multiplex genome engineering using CRISPR/Cas systems. Science 339, 819-823.
```

### Author Rules
- List up to 10 authors; if more than 10, list first 10 followed by "et al."
- Use initials after surname: Smith, J.A.
- Last author preceded by "and": Author1, A., Author2, B., and Author3, C.
- For two authors: Author1, A., and Author2, B.

### Book Chapters
```
Author, A.A. (Year). Chapter title. In Book Title, Editor, A.A., ed. (Publisher), pp. pages.
```

### Books
```
Author, A.A. (Year). Book Title (Publisher).
```

### Preprints
```
Author, A.A. (Year). Title. Preprint at server. DOI.
```
Example:
```
Smith, J., and Jones, K. (2023). Novel findings in cell biology. Preprint at bioRxiv. https://doi.org/10.1101/2023.01.01.123456
```

### Datasets
```
Author, A.A. (Year). Dataset title. Repository. Accession/DOI.
```

### Software/Code
```
Author, A.A. (Year). Software name (version). URL or DOI.
```

### Web Resources
```
Organization (Year). Title. URL.
```

## Formatting Rules

### Year
- Always in parentheses: (2023)

### Volume and Pages
- Volume in bold (in final typeset) but plain in manuscript
- Pages separated by en-dash: 123-145
- Article numbers acceptable when no page numbers: e107892

### DOI
- Include DOI for all references when available
- Format: https://doi.org/10.xxxx/xxxxx

## Special Cases

### In Press / Accepted
```
Author, A.A. (2024). Title. Journal, in press. DOI if available.
```

### Published Online / Ahead of Print
```
Author, A.A. (2024). Title. Journal. Published online Month Day, 2024. DOI.
```

### Retracted Papers
- Do not cite retracted papers as supporting evidence
- If essential to cite, note the retraction clearly

### Personal Communications
- Avoid; if necessary, cite in text only (not in reference list)
- Requires written permission from the person cited

## Validation Checklist

- [ ] All citations numbered sequentially in order of appearance
- [ ] No gaps in citation numbering
- [ ] All reference list entries have corresponding in-text citations
- [ ] No orphan references (in list but not cited in text)
- [ ] Author names formatted correctly (Surname, Initials.)
- [ ] Year in parentheses
- [ ] Journal names correctly formatted
- [ ] DOIs included where available
- [ ] "et al." used correctly (>10 authors)
- [ ] Preprints clearly labeled
- [ ] No retracted papers cited as supporting evidence
- [ ] Page ranges use en-dash
- [ ] Reference list entries end with period

## Common Errors

| Error | Correction |
|-------|-----------|
| Missing "and" before last author | Add "and" before final author |
| Abbreviating Cell Press journal names | Use full journal name for Cell Press titles |
| Using brackets [1] instead of superscript | Cell Press uses superscript numbers |
| Missing DOI | Add DOI for all available references |
| Inconsistent date format | Always use (Year) format |
| Citing >10 authors without et al. | List first 10, then et al. |
| Missing period at end of reference | All entries end with period |
